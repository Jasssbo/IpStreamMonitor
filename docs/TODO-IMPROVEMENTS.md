# Parametri utili per una radio/webradio

---

## Parametro->effetto

```text
Bitrate effettivo -> Qualità streaming, stabilità connessione
Dropout/Silence count -> Rilevare interruzioni o "buchi" audio
Tempo dall'ultima riconnessione -> Stabilità
Jitter (σ latenza) -> Variabilità connessione
Clipping events -> Campioni a 0dBFS = distorsione
Correlazione stereo -> Rilevare mono mascherato da stereo
Dynamic Range -> Per valutare compressione eccessiva
Peak-to-LUFS ratio -> Crest factor ~ compressione.
Loudness Range (LRA) -> Variabilità dinamica (EBU R128)
```
